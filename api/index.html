Barrister Brief API

This API features endpoints which allows for programmatic access to our data.

Endpoints are described below.

Name Age
"/case" Must be called with at least one of the following query parameters:
  • citation - fetch a specific hearing (not usuable with other options)
  • start & end - a date range in YYYY-MM-DD format (must provide both)
  • favour - filter by verdict favour (must be plaintiff or defendent)
"/judge" Returns all judges
"/judge/[judge-id]" Returns data about judge with via judge-id
"/judge/[judge-id]/case" Get all hearings sat by judge-id
